# Bienvenido al Curso SQL

Curso completo de SQL organizado en 5 módulos progresivos. Cubre desde los conceptos fundamentales de bases de datos relacionales hasta consultas avanzadas multitabla, con ejercicios resueltos en cada etapa.

## Camino de aprendizaje

La documentación está organizada en módulos progresivos. Se recomienda seguirlos en orden:

<Steps>
  <Step title="Fundamentos" icon="database">
    **Entendé qué son las bases de datos.** Modelos de BDD, tablas, campos, registros, claves primarias y foráneas, relaciones entre entidades y diagramas ER.
  </Step>
  <Step title="SQL Básico" icon="code">
    **Aprendé a consultar y manipular datos.** SELECT, WHERE, tipos de datos, CREATE, ALTER, DROP, INSERT, UPDATE, DELETE, índices, vistas y procedimientos almacenados.
  </Step>
  <Step title="Consultas Intermedias" icon="layer-group">
    **Dominá filtros avanzados y funciones.** LIKE, BETWEEN, IN, ORDER BY, GROUP BY, HAVING, CASE, funciones agregadas, de cadena, de fecha y subconsultas.
  </Step>
  <Step title="JOINs" icon="link">
    **Combiná datos de múltiples tablas.** JOIN implícito, INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN, JOINs multitabla y UNION.
  </Step>
  <Step title="SQL Avanzado" icon="graduation-cap">
    **Consultas complejas y seguridad.** Subconsultas como origen de tabla, consultas multitabla avanzadas, inyección SQL y herramientas de gestión.
  </Step>
</Steps>

<AccordionGroup>
  <Accordion title="Glosario de términos" icon="book">

| Término | Definición |
|---------|-----------|
| **BDD** | Base de datos |
| **RDBM** | Sistema de gestión de bases de datos relacional |
| **DBMS** | Sistema de gestión de bases de datos (Database Management System) |
| **DDL** | Data Definition Language — CREATE, ALTER, DROP |
| **DML** | Data Manipulation Language — INSERT, UPDATE, DELETE, SELECT |
| **PK** | Primary Key — Clave primaria, identifica de forma única cada registro |
| **FK** | Foreign Key — Clave foránea, referencia a la PK de otra tabla |
| **JOIN** | Operación que combina filas de dos o más tablas basándose en una columna relacionada |
| **Tupla** | Una fila o registro de una tabla |
| **Collation** | Reglas de comparación de cadenas de texto en una base de datos |

  </Accordion>
</AccordionGroup>
